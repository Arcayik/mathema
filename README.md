Mathema is a CLI calculator that supports user-defined variables and functions, with future plans to support solving algebraic expressions, differentiaton, integration, and more!

## Features

* Operators: `+`, `-`, `*`, `/`, `^`
* Variables: `x = 5`
* Functions: `f(x) = 2*x + 1`
* Built-in Functions and Variables: `sin(x)`, `log(x)`, `pi`, etc.

## Roadmap

- [ ] Manual Algebra Tree Transforms
- [ ] Equality Solver
- [ ] Differentiation
